# Markdown Cheat Sheet

### Heading

```text
# Title Large
## Title Medium
### Title Small
```

### Bold

```text
**bold**
```

### Italic

```text
*italic*
```

### Blockquote

```text
> blockquote
```

### Ordered list

```text
1. Alpha
2. Bravo
3. Charlie
```

### Unordered list

```text
- Alpha
- Bravo
- Charlie
```

### Code

```text
This is `inline` code
```

```text
```
multiple
lines of code
```
```

### Horizontal rule

```text
---
```

### Link \(URL\)

```text
[title](https://tasksift.com)
```

### Image

```text
![alt text](https://example.com/images/cat.jpg)
```
